# Experiment 3.1 – Product Card Component

## Objective
- Design a responsive product card
- Implement props for dynamic data
- Style with modern CSS techniques
- Display stock status visually

## Description
A reusable `ProductCard` component is created using React props.  
The component dynamically displays product information such as:

- Product name
- Price
- Description
- Stock availability

Multiple product cards are rendered using the same component.

# Experiment 3.2 – Library Management System

## Objective
- Create responsive book listing
- Implement search functionality
- Add form for new book entries
- Enable book removal
- Manage state using React hooks

## Description
This module simulates a simple library management system.  
Users can:

- Add new books
- Search for books
- Remove books

State is managed using the `useState` hook.

# Experiment 3.3 – Class Hierarchy with Inheritance

## Objective
- Create base `Person` class
- Extend with `Student` and `Teacher` subclasses
- Implement method overriding
- Demonstrate polymorphism
- Render class instances in UI

## Description
This module demonstrates object-oriented programming concepts using JavaScript classes.

The hierarchy includes:

Person  
├── Student  
└── Teacher  

Each subclass overrides methods from the base class to demonstrate polymorphism.
